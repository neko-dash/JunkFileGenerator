# 1. 概要
#========================================================================================================

    1.1. はじめに
        「JunkFileGenerator」は、ランダムな内容のファイルを生成するアプリケーションソフトです。

        パソコン、SSD、ハードディスク、USBメモリなどを廃棄・譲渡する際に、記録領域を
        ジャンクファイルで上書きすることで、元のデータを完全に復元できないようにします。

        作者は、既存の完全消去ツールと併用する補助的な用途として、このアプリケーションを利用しています。
        入手した記憶媒体が最初からジャンクファイルで満たされていれば、悪意ある第三者も
        ファイルの復元を最初から諦める可能性があります。

        意外にも、このような用途に適した使いやすいソフトが世の中にほとんど見当たらなかったため、
        勉強を兼ねて自作しました。

        本ソフトは、Linux、Windows、macOS などの複数のOSでの利用を想定し、Pythonで設計・実装されています。

# 2. 使用方法
#========================================================================================================

    2.1. 起動方法
        方法 1 : 「JunkFileGenerator.py」を任意のフォルダに保存し、以下のコマンドで起動します。

            python3 JunkFileGenerator.py

            ※ 事前に Python がインストールされている必要があります。

        方法2 : 各OS用の実行ファイルをダブルクリックして起動します。(1.2項参照)

    2.2. 使い方
        起動後、以下の項目を設定して "Generate" ボタンを押してください。

        --------------------+------------------------------------------------------------------------
         設定名             | 設定内容
        --------------------+------------------------------------------------------------------------
         Fill entire drive  | ドライブの空き容量すべてをジャンクファイルで満たす場合にチェックを入れます。
                            | チェックを入れると "Number of files" の設定は無効になります。
        --------------------+------------------------------------------------------------------------
         File copy mode     | 最初にジャンクファイルを1つ生成し、それをコピーしてファイルを増やす場合に
                            | チェックを入れます。生成し続けるよりも処理が速くなる可能性があります。
                            | 空き容量が "File size" 未満になると、コピーから再生成に切り替わります。
        --------------------+------------------------------------------------------------------------
         File size          | 1ファイルあたりのサイズを指定します。
                            | 単位は Gbyte、Mbyte、Kbyte、byte から選択可能です。
        --------------------+------------------------------------------------------------------------
         Number of files    | 生成するファイル数を指定します。
                            | "Fill entire drive" にチェックが入っている場合は無効になります。
        --------------------+------------------------------------------------------------------------
         Output folder      | ファイルの出力先フォルダを指定します。
        --------------------+------------------------------------------------------------------------

        記録媒体を限界までジャンクファイルで満たして、安全なデータ消去をお試しください。
